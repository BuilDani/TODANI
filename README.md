TaskFlow (working title)
Overview

TaskFlow is an Android app built with Kotlin in Android Studio, currently at an early stage of development. The app is designed to run exclusively on Android devices. Its main goal is to help users stay on top of their daily routines by providing a smart task management system with recurring tasks, reminders, and integrations.

The focus is not just on simple to-do lists, but on habit reinforcement and memory support. By allowing tasks to automatically reset every midnight, the app ensures that important activities are not forgotten and can be repeated daily, weekly, or on a custom schedule.

Core Features

Recurring Task Groups
Create sets of tasks that automatically reset at midnight, helping users stay consistent with daily habits.

Custom Reminders
Add reminders for specific times, such as “take out the trash every Wednesday before 9:30.”

Calendar Integration
Sync tasks seamlessly with Google Calendar or a standard calendar for a unified view of schedules.

Planned Features

AI-Powered Routines (with Payment Wall)
Users will be able to request personalized routines, such as workout plans or study schedules. Accessing this feature will prompt a payment wall, unlocking premium AI-driven assistance.

Expense Tracking
A financial tracking module inspired by Completa.ai, allowing users to log and monitor expenses alongside their tasks.

Tech Stack

Language: Kotlin

IDE: Android Studio

Platform: Android (exclusive)

Development Status

This project is still in its early development phase. The core functionality (recurring tasks and reminders) is the first priority, with AI integration and expense tracking to follow in later stages.
